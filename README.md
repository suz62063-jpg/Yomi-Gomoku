Yomi Gomoku

一个为浏览器设计的五子棋 AI。

基于 Alpha-Beta Search、Zobrist Hashing 与迭代加深实现，
尝试在轻量化的前提下，
提供一种更安静、更自然的对弈体验。

⸻

Features

* Alpha-Beta 搜索
* Iterative Deepening
* Zobrist Hashing
* Threat Evaluation
* Pure Web Implementation
* Minimal Interface

⸻

Philosophy

这个项目并不追求“最强”。

相比算力，
它更在意：

* 落子的节奏
* 思考的停顿
* UI 的克制感
* 人与 AI 对弈时的氛围

希望它看起来不像传统网页游戏。

而更像一个安静坐在你对面的棋手。

⸻

Run

直接打开：

Yomi Gomoku.html

无需安装环境，
无需额外依赖。

⸻

Notes

有时候，
它会下得出奇地认真。

甚至让人忘记，
这只是一个运行在浏览器里的小型 AI。
